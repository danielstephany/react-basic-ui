## Automatic column widths

If a width option is not set for a column, the column will grow to the size of its content or fill the remaining space in the row if there is not enough content to fill the row.

## Simple grid layout without breakpoints

To Produce a simple responsive grid that maintains the same layout across all screen sizes pass only the <strong>xs</strong> prop to the columns with the desired value.

    <Story name="Basic Grid">
